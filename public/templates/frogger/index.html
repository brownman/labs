Requires a browser that supports the HTML5 Canvas element, pretty much all modern browsers besides Internet Explorer.

Technology Used

Cake.js - A Javascript canvas animation/gaming framework. Helpful in abstracting some of the lower level canvas manipulation out from this game's code.

Facebook Graph API - Using simple javascript implementation of Facebook connect to authenticate people who play and want to record their score to the leaderboard or share with friends.
